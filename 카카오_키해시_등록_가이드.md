# 카카오 개발자 콘솔에 키 해시 등록하기

## 1. 키 해시 확인 방법

### 방법 A: 앱 실행 후 Logcat에서 확인 (권장)

1. Android Studio를 열고 프로젝트를 엽니다
2. 앱을 실행합니다 (`npx expo run:android` 또는 Android Studio에서 실행)
3. Android Studio 하단의 **Logcat** 탭을 엽니다
4. 필터에 `KeyHash`를 입력합니다
5. 다음과 같은 로그가 출력됩니다:
   ```
   KeyHash: =========================================
   KeyHash: 카카오 개발자 콘솔에 등록할 키 해시:
   KeyHash: [여기에 키 해시가 표시됩니다]
   KeyHash: =========================================
   ```
6. 키 해시를 복사합니다 (공백 제거)

### 방법 B: 명령어로 확인

터미널에서 다음 명령어를 실행하세요:

```bash
# macOS/Linux
keytool -exportcert -alias androiddebugkey -keystore ~/.android/debug.keystore -storepass android -keypass android | openssl sha1 -binary | openssl base64

# Windows (Git Bash 사용)
keytool -exportcert -alias androiddebugkey -keystore %USERPROFILE%\.android\debug.keystore -storepass android -keypass android | openssl sha1 -binary | openssl base64
```

## 2. 카카오 개발자 콘솔에 등록하기

1. [카카오 개발자 콘솔](https://developers.kakao.com/)에 로그인합니다
2. 내 애플리케이션 > 앱 선택 > 앱 설정 > 플랫폼 메뉴로 이동합니다
3. **Android 플랫폼**을 선택합니다
4. **키 해시** 입력란에 위에서 복사한 키 해시를 붙여넣습니다
5. **저장** 버튼을 클릭합니다

## 3. 주의사항

- **디버그 키 해시**: 개발 중에는 디버그 키 해시를 등록합니다
- **릴리즈 키 해시**: 앱을 배포할 때는 릴리즈 키스토어의 키 해시도 등록해야 합니다
- **키 해시 형식**: 공백 없이 한 줄로 입력해야 합니다
- **여러 키 해시 등록 가능**: 디버그와 릴리즈 키 해시를 모두 등록할 수 있습니다

## 4. 릴리즈 키 해시 확인 방법

릴리즈 키스토어가 있는 경우:

```bash
keytool -exportcert -alias [키스토어 별칭] -keystore [키스토어 경로] -storepass [비밀번호] -keypass [비밀번호] | openssl sha1 -binary | openssl base64
```

## 5. 문제 해결

- 키 해시가 맞는데도 로그인이 안 되는 경우:
  - 카카오 개발자 콘솔에서 키 해시가 정확히 등록되었는지 확인
  - 앱을 완전히 종료하고 다시 실행
  - 카카오톡 앱이 설치되어 있는지 확인
  - 네트워크 연결 확인
