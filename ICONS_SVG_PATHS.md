## 📏 아이콘 사용 규칙

### 크기 가이드
- **소형**: `w-4 h-4` (16px) - 작은 버튼, 태그
- **중형**: `w-5 h-5` (20px) - 기본 아이콘
- **대형**: `w-6 h-6` (24px) - 주요 아이콘
- **특대**: `w-8 h-8` (32px) - 강조 아이콘
- **초대형**: `w-10 h-10` (40px) - 프로필, 카테고리

### 스트로크 가이드
- **기본**: `strokeWidth={2}` - 대부분의 아이콘
- **얇게**: `strokeWidth={1.5}` - 섬세한 표현
- **두껍게**: `strokeWidth={2.5}` - 강조

### 색상 가이드
- **단색**: `text-white`, `text-gray-700` 등
- **그라데이션 배경 위**: 대부분 `text-white`
- **밝은 배경 위**: `text-gray-700` 또는 그라데이션 색상

## 🔧 커스텀 아이콘 만들기

새로운 커스텀 SVG 아이콘이 필요한 경우:

1. **Figma에서 디자인**
2. **SVG로 내보내기**
3. **`/imports/svg-[고유ID].ts` 파일 생성**
4. **Path 데이터 추출 및 저장**
5. **컴포넌트에서 import하여 사용**

### 예시 구조

```typescript
// /imports/svg-custom123.ts
export default {
  pathId1: "M10 20 L20 10...",
  pathId2: "M5 5 C10 10...",
}
```

```tsx
// 컴포넌트에서 사용
import customPaths from './imports/svg-custom123';

<svg>
  <path d={customPaths.pathId1} stroke="white" />
  <path d={customPaths.pathId2} stroke="white" />
</svg>
```
